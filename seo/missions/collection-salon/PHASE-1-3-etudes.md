- **Pas de cannibalisation nuisible** entre la collection salon et les autres **collections**.

## PHASE 2 — Décodage de la SERP « tableau salon » (FR)

**Format dominant** : pages **collection e-commerce** (catégorie transactionnelle), pas d'éditorial pur en top. Présence de gros acteurs (Izoa, Hexoa, Tablodeco, toiles-modernes, amesauvage, Uwalls, Posterlounge) + retailers généralistes (Leroy Merlin, Vente-unique, Manomano).

**Ce que Google récompense (constantes du top) :**
- H1 riche en keyword + modificateur de style (« moderne / tendance / chic »).
- Bloc(s) de **texte éditorial** (300-500+ mots) — souvent intro courte en haut + développement.
- **Hiérarchie Hn** par **style** (paysage, abstrait, moderne, zen, panoramique…) et par **conseils** (taille, couleurs, emplacement).
- **USP** visibles : matériaux (toile, cadre bois), livraison, satisfait/remboursé, SAV, avis.
- **Maillage interne dense** (autres pièces, styles, formats, couleurs).
- Rich snippets : **Breadcrumb** + **Product** (étoiles avis chez Izoa). **FAQPage quasi absent.**

**Champ sémantique attendu** : moderne, tendance, chic, design, contemporain, abstrait, zen, minimaliste, paysage, coloré, original ; tableau / toile / **cadre** / mural / peinture / impression ; décorer / habiller / sublimer / aménager ; canapé, mur, ambiance, atmosphère, cocooning ; formats triptyque / XXL / panoramique ; matières toile / bois.
